IT TERMS
• 1.Backup: Fazer uma cópia de programas ou dados para evitar perda dos mesmos.

• 2.BIOS: BIOS é a sigla para Basic Input/Output System, ou Sistema Integrado de Entrada e Saída.

• 3.Browser: Browser ou Navegador é um programa criado para permitir a navegação pela internet.

• 4.Bug: O termo “Bug” é um jargão usado no ambiente de desenvolvimento para identificar uma falha no sistema.

• 5.Copyright: Copyright é uma maneira de proteger materiais autorais, dando direito de propriedade a quem 
utiliza o intelecto para produzir algo único.

• 6.Crack: É um programa criado para violar outros programas sem permissão do autor. 
Ex: programas piratas craqueados, usados sem direito, ilegalmente.
Um crack é um pequeno software usado para quebrar um sistema de segurança qualquer.

• 7.Data: Dados/Informações.

• 8.Database: Banco de Dados.

• 9.Full screen: Tela cheia/Completa.

• 10.Freeware: Programa gratuito, que baixamos da internet.

• 11.Features: Características.

• 12.Home page: Página principal.

• 13.Input: Entrada.

• 14.Join: Unir-se/Juntar-se.

• 15.Keyboard: Teclado de computador.

• 16.Keyword: Palavra-chave.

• 17.Log in/on: Iniciar a sessão/Conectar-se a algo.

• 18.Lay out: Planta de um projeto.

• 19.Log of / log out: Encerrar a sessão/Desconectar-se de algo.

• 20.Load: Carregar.
